# Thrive
A dynamic bucket list web application that helps users set, track, and achieve their life goals. Create personalized lists, celebrate milestones, and thrive by turning dreams into reality, one check at a time.

## Features

- Create, edit, and delete bucket list items  
- Mark items as completed  
- Track progress and milestones  

---

## Tech Stack

- **Frontend:** Angular  
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Package Manager:** npm or Yarn  

---

## Getting Started

Follow these steps to set up the project locally:

### 1. Clone the repository
```bash
git clone https://github.com/rphan01/Thrive.git
cd Thrive
```

### 2. Install dependencies
```bash
npm install
# or
yarn install
```

### 3. Run the application
```bash
ng serve
```
